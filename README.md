# Durability Alert

This Minecraft Spigot Plugin adds alerts that warn's Players when their Armor/Tools are about to break



## Features

- Alerts when Armor or Tools are about to break
- Fully customizable
- Chat alerts
- Title alerts
- Subtitle alerts
